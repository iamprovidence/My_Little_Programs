# Text Rephraser

A React application that rephrases text using **Google Gemini AI**.

## Setup Instructions

### 1. Install Dependencies

```bash
npm install
```

### 2. Get Your FREE Google Gemini API Key

Get your **free** API key from [Google AI Studio](https://aistudio.google.com/app/apikey).

**Steps to get your API key:**
1. Go to https://aistudio.google.com/app/apikey
2. Sign in with your Google account
3. Click "Create API key"
4. Select "Create API key in new project"
5. Copy the API key (starts with `AIza`)
6. Enter it in the app when you first run it

**Why Google Gemini?**
- ✅ **Completely FREE** - No credit card required
- ✅ **1,500 requests per day** for free
- ✅ **High quality** - Comparable to ChatGPT
- ✅ **Fast** responses

**Important Security Notes:**
- Your API key is stored locally in your browser (localStorage)
- Never share your API key publicly


### 3. Run the Development Server

```bash
# Double-click: start.cmd
# OR run in terminal:
start.cmd
```


The app will open at `http://localhost:5173`
